How to Use Zamini PDF Converter V1.1.0

Getting Started

  1. Click Add Files or Drag & Drop to select files.
  2. Type a file name for the PDF (default: date & time).
  3. Click Single PDF to merge all files.
  4. Click Separate PDFs to convert each file individually.
  5. Track progress using the progress bar.

File Management

  • Move Up / Move Down — change file order
  • Delete Selected — remove files
  • Reset — clear all files

Requirements

  • Microsoft Office — required for Word, Excel, PowerPoint

Supported File Types

  • PDF: .pdf
  • Images: .jpg, .jpeg, .png, .webp, .bmp, .tiff, .gif, .ico, .jfif
  • Word: .docx, .doc
  • Excel: .xlsx, .xls
  • PowerPoint: .pptx, .ppt
  • Text: .txt, .log, .csv, .tsv, .md
  • Code: .py, .c, .cpp, .java, .js, .sh, .bat
  • Config/Data: .json, .xml, .yaml, .yml, .ini, .conf
  • Web: .html, .css
